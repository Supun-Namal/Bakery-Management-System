# Bakery Management System
Fresh Slice 5 is a comprehensive Java-based management solution designed for bakery operations. 
It provides modules for user management and product tracking through an intuitive graphical interface.

# Features

* User Authentication: Secure login system for authorized access.
* Product Management: Tools to add, update, and manage bakery products.
* User Management: Administrative controls to manage system users and roles.
* MVC Architecture: Organized code structure separating data (Model), interface (View), and logic (Controller).

# Project Structure

* src/controller/: Contains logic for handling user inputs and system flow (e.g., LoginController, ProductController).
* src/view/: Contains the GUI components and forms (e.g., LoginView, UserManagementView).
* src/model/: Defines data structures and business logic.
* src/database/: Handles database connections (e.g., DBConnection).
* src/Images/: Stores graphical assets like logos and icons.


# Getting Started

## Prerequisites

* Java Development Kit (JDK): Version 8 or higher.
* IDE: NetBeans (Recommended) or IntelliJ IDEA.
* Database: MySQL (Ensure the database is running as per the DBConnection settings).

# Note

This project was developed to demonstrate core Object-Oriented Programming (OOP) principles,
Graphical User Interface (GUI) design, and the Model-View-Controller (MVC) design pattern.
